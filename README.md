# Universal-Aether

Bu depo `sgy_format.py` calistiricisini ve onun bootstrap komutunu icerir.

## Calistirma

```bash
python3 master_run.py
```

`master_run.py` artik yalnizca `requirements.txt` icinde gercek bir ucuncu taraf bagimliligi varsa `pip install` dener. Bos veya sadece yorum iceren dosyalarda kurulum adimi atlanir.
